# Yamblz-Weather
### Реализовано:
- 2-pane
- автообновление погоды
- выбор города с подсказками
- избранные города
- удаление из избранного
- navigation drawer
- прогноз погоды на некоторое время вперед (OpenWeatherMap API)
- ci
- unit и ui тесты
- мультиязычность

### Используемые инструменты:
- RxJava/RxBinding
- Dagger 2
- Retrofit 2
- Moxy
- Android-job
- ButterKnife
- AdapterDelegates
- MaterialDrawer
- Jacoco
- Mockito
- Travis ci
- Stetho
- LeakCanary
- GreenDao ORM
